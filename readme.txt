To run the database;
1) Run new.sql in SQL WORKBENCH to get the tables

To start the server:
1) `cd server`
2) `nodemon server`

To start the frontend index.js
1) `cd Frontend`
2) `nodemon index`

Or alternatively, you can start both of them at the same time by running
1) `npm run dev`
in the root folder

To look at the api's available head to api.docx
